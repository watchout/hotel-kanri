## 🛠️ **Phase 2: 実装実行フェーズ**

### **2.1 指示書確認・実装準備**
```yaml
実装前チェック:
  ✅ Izaの指示書内容確認
  ✅ 参照仕様書の事前読み込み
  ✅ 制約事項の理解
  ✅ 完了条件の明確化

準備完了の確認:
  - 必要な仕様書は全て確認済みか
  - システム間制約は理解したか
  - 品質基準は明確か
  - 実装手順は具体的か
```

### **2.2 段階的実装実行**
```yaml
実装フロー:
  1. 指示書Step 1実行
  2. 中間確認・テスト
  3. 指示書Step 2実行
  4. 中間確認・テスト
  5. 指示書Step 3実行
  6. 最終確認・統合テスト

各段階での確認事項:
  - 指示書通りの実装か
  - 品質基準を満たしているか
  - システム間制約を遵守しているか
  - 予期しない副作用はないか
```
