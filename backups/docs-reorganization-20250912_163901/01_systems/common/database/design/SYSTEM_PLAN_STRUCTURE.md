# ホテル統合システムのプラン構造

## システム構成

ホテル統合システムは以下のサブシステムで構成されています：

```
ホテル統合システム
├── AIコンシェルジュ (hotel-saas)
├── ホテルマネジメントシステムPMS (hotel-pms)
└── ホテル顧客管理システムCRM (hotel-member)
```

## プラン体系

各サブシステムは、業態別・プラン種別の組み合わせで提供されます。

### 業態区分

- **一般ホテル向け** (general)
- **レジャー向け** (leisure)
- **海外向け** (overseas)

### プラン種別

- **エコノミー** (economy)
- **プロフェッショナル** (professional)
- **エンタープライズ** (enterprise)
- **アルティメット** (ultimate)

## 注意事項

- 各システムは独立して契約可能です
- 複数システムの契約には割引が適用されます
- 年間契約の場合は月額料金の10%割引となります
- 各プランには初期設定費用が別途必要です
