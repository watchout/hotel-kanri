## 使い方

1. Linearで「New issue」をクリック
2. 「Choose template」で「Bug Fix Task」を選択
3. タイトルを `[バグ修正] 簡潔な説明` に設定
4. Priority を Severity に応じて設定
5. Labels に「bug-fix」を追加
6. 「Create issue」
