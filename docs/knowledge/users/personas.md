# ユーザーペルソナ - Personas

> 最終更新: 2026-02-05

---

## 1. ホテルオーナー / 経営者

| 項目 | 内容 |
|:-----|:-----|
| 役割 | システム導入決定者、全体管理 |
| 目的 | 業務効率化、顧客満足度向上、コスト削減 |
| 懸念 | 導入コスト、スタッフの習熟、セキュリティ |
| 使用機能 | ダッシュボード、統計レポート、スタッフ管理 |

---

## 2. フロントデスクスタッフ

| 項目 | 内容 |
|:-----|:-----|
| 役割 | 日常オペレーション、ゲスト対応 |
| 目的 | 業務効率化、ゲスト対応品質向上 |
| 懸念 | 操作の複雑さ、システム障害時の対応 |
| 使用機能 | 注文管理、ハンドオフ対応、デバイス管理 |

## 3. ホテルゲスト

| 項目 | 内容 |
|:-----|:-----|
| 役割 | サービス利用者 |
| 目的 | 快適な滞在、スムーズなサービス利用 |
| 懸念 | 言語の壁、操作の分かりやすさ |
| 使用機能 | メニュー閲覧、注文、AIコンシェルジュ |
