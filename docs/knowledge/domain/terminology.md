# 業界用語 - Terminology

> 最終更新: 2026-02-05

## ホテル業界用語

| 用語 | 英語 | 定義 |
|:-----|:-----|:-----|
| テナント | Tenant | システムを利用するホテル事業者 |
| ゲスト | Guest | ホテルの宿泊客 |
| ルームグレード | Room Grade | 客室のグレード（スタンダード、デラックス等） |
| デバイス | Device | 客室に設置されたタブレット等の端末 |
| ハンドオフ | Handoff | AI対応からスタッフ対応への引継ぎ |
| コンシェルジュ | Concierge | 宿泊客へのサービス案内担当 |
