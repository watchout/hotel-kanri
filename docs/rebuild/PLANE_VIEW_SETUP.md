## 🚀 作成手順

### Step 1: Planeを開く

以下のURLを開いてください：

```
https://plane.arrowsworks.com/co/projects/7e187231-3f93-44cd-9892-a9322ebd4312/issues
```

---

### Step 2: ビュー「Ready to Start」を作成

#### 2-1. フィルターを開く

1. 画面上部の **「Filters」** ボタンをクリック
2. または、右上の **「⚙️ (設定アイコン)」** → **「Filters」**

#### 2-2. フィルター1を追加

1. **「+ Add Filter」** をクリック
2. フィルター設定:
   - **Property**: `State`
   - **Condition**: `is`
   - **Value**: `Backlog`

#### 2-3. フィルター2を追加

1. **「+ Add Filter」** をクリック（再度）
2. フィルター設定:
   - **Property**: `Description`
   - **Condition**: `does not contain`
   - **Value**: `Blocked by`

#### 2-4. ソート設定

1. 画面上部の **「Sort」** ボタンをクリック
2. ソート設定:
   - **Property**: `Priority`
   - **Order**: `High → Low` (降順)

#### 2-5. ビューを保存

1. 画面右上の **「Save View」** または **「💾」** アイコンをクリック
2. ビュー名を入力:
   ```
   Ready to Start
   ```
3. **「Save」** をクリック

---

### Step 3: ビュー「Blocked」を作成

#### 3-1. 新しいビューを開始

1. 画面左側の **「Views」** セクション
2. **「+ New View」** をクリック

#### 3-2. フィルターを追加

1. **「+ Add Filter」** をクリック
2. フィルター設定:
   - **Property**: `Description`
   - **Condition**: `contains`
   - **Value**: `Blocked by`

#### 3-3. ソート設定

1. 画面上部の **「Sort」** ボタンをクリック
2. ソート設定:
   - **Property**: `Priority`
   - **Order**: `High → Low` (降順)

#### 3-4. ビューを保存

1. 画面右上の **「Save View」** をクリック
2. ビュー名を入力:
   ```
   Blocked
   ```
3. **「Save」** をクリック

## 🚨 トラブルシューティング

### 問題1: 「Filters」ボタンが見つからない

**対処法**:
- 画面右上の **「⚙️」** アイコンをクリック
- または、キーボードショートカット: `F` キーを押す

### 問題2: 「Description」フィールドが見つからない

**対処法**:
- フィルター追加時、検索ボックスに `Description` と入力
- または、スクロールして探す

### 問題3: ビューに何も表示されない

**原因**: フィルター条件が厳しすぎる

**対処法**:
1. フィルターを一時的に削除
2. 「All Issues」ビューに戻る
3. 再度フィルターを追加
