## 1. 概要

### 1.1 目的

レジャーホテル・旅館（20-50室）の経営者・マネージャーにペインを訴求し、リード獲得（資料請求・診断申込）につなげる。

### 1.2 KPI

| 指標 | 目標 |
|:-----|:-----|
| CVR（訪問→CV） | 5%+ |
| 直帰率 | 40%以下 |
| 平均滞在時間 | 3分+ |
| ページ速度（LCP） | 2.5秒以内 |

### 1.3 関連ドキュメント

| ドキュメント | 用途 |
|:-------------|:-----|
| `SSOT_LEAD_ACQUISITION.md` | リード獲得戦略 |
| `SSOT_MARKETING_STRATEGY.md` | マーケティング全体戦略 |
| `SSOT_PRICING_ENTITLEMENTS.md` | 料金プラン |

### 6.3 CRM連携（HubSpot）

```typescript
// server/utils/hubspot.ts

import { Client } from '@hubspot/api-client'

const hubspot = new Client({ accessToken: process.env.HUBSPOT_ACCESS_TOKEN })

export async function createContact(data: {
  email: string
  name: string
  phone?: string
  company?: string
}) {
  return await hubspot.crm.contacts.basicApi.create({
    properties: {
      email: data.email,
      firstname: data.name.split(' ')[0],
      lastname: data.name.split(' ')[1] || '',
      phone: data.phone || '',
      company: data.company || ''
    }
  })
}

export async function createDeal(contactId: string, facilityName: string) {
  return await hubspot.crm.deals.basicApi.create({
    properties: {
      dealname: `${facilityName} - LP問い合わせ`,
      dealstage: 'appointmentscheduled',
      pipeline: 'default'
    },
    associations: [
      {
        to: { id: contactId },
        types: [{ associationCategory: 'HUBSPOT_DEFINED', associationTypeId: 3 }]
      }
    ]
  })
}
```
